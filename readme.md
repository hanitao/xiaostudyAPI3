# 说明
~~~
一个基于maven的工具类包
~~~

# 工具类

### 1、日期

- 处理日期格式，转化不同格式；
- 两个日期的时间差

### 2、文件

- 复制/移动文件/文件夹
- 判断文件是否为指定某类个/某些文件
- 统计文件夹内文件夹个数和文件个数
- 获取文件大小
- 把某个/某些文件复制到指定文件夹下
- 删除指定类型文件

### 3、Excel

- 读取和写入Excel，支持xls和xlsx格式

### 4、数学

- 排列组合
- 阶乘
- 一元二次方程
- 等差等比数列
- 海伦公式
- 二项分布
- 两个数的最大公约数
- 小数转分数（精度丢失下失效）
